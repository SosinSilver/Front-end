## 설치
### Frontend

- Clone Project
```
git clone https://lab.ssafy.com/swsilver95/final-pjt-front.git
```

- Project setup
```
cd final-pjt-front
npm install
```

- run frontend server
```
npm run serve
```


### Backend

- Clone Project
```
git clone https://lab.ssafy.com/swsilver95/final-pjt-back.git
```

- Project setup
```
cd final-pjt-back
pip install -r requiements.txt
```

- run backend server
```
python manage.py runserver
```
